# 介绍
AQQBot-web是专门为AQQBot设计的后台面板，用于管理AQQBot的插件、配置、玩家数据等  
此面板是独立于AQQBot插件之外的项目，开发者为 *mc506lw*

## 功能
- **管理绑定**: 可以管理玩家的绑定关系
- **多端管理**: 可以管理多个AQQbot插件
